## 快速開始

### 第一步：安裝擴充功能

1. [下載最新版 ZIP](https://github.com/chenmitchell/social-buddy-release/raw/main/socialbuddy-v1.0.3.zip)
2. 解壓縮到一個資料夾（建議放在不會誤刪的地方）
3. Chrome 瀏覽器打開 `chrome://extensions/`
4. 右上角開啟「開發人員模式」
5. 點「載入未封裝項目」，選擇解壓縮出來的資料夾

### 第二步：釘選到工具列

安裝完成後，點 Chrome 工具列的拼圖圖示 🧩，找到 SocialBuddy，按釘選圖示把它固定在工具列上，方便隨時使用。

### 第三步：開始瀏覽

打開 Threads、Instagram 或 Facebook，SocialBuddy 會在背景自動開始備份你看到的貼文。點工具列上的 SocialBuddy 圖示就能開啟側邊面板，查看已備份的內容。

## 認識介面

SocialBuddy 主要有兩個操作區域：

**側邊面板（Side Panel）**
點擊工具列圖示後開啟的面板，這是你的主要工作區。可以在這裡瀏覽備份的貼文、搜尋、匯出報告、使用快速回覆。面板上方有搜尋框和篩選標籤，下方是貼文列表。

**設定頁面（Options）**
右鍵點選工具列圖示 → 選項，可以進入設定頁面。在這裡管理關鍵字、排程、匯出偏好、快速回覆範本等進階設定。

## 情境教學：社群小編海巡的一天

> 小美是一家電商品牌的社群小編，負責經營品牌的 Threads、IG 和 FB 粉專。每天早上她都要巡一輪各平台，處理客訴、觀察競品、保護品牌圖片不被盜用。以下是她用 SocialBuddy 工作的一天。

---

### 早上 9:00 — 開工，打開巡邏模式

小美打開 Chrome，先逛一圈品牌的 FB 粉專和 IG 頁面，看看昨晚有沒有新的留言或貼文。

她不用做任何操作，SocialBuddy 在背景就會自動把她瀏覽到的貼文存下來。每一則貼文的文字、圖片、留言數、按讚數、發布時間，全部自動記錄。

**她做了什麼：** 就是正常瀏覽，什麼都不用多做。
**SocialBuddy 做了什麼：** 自動把所有瀏覽到的貼文備份到本機資料庫。

---

### 早上 9:30 — 設定關鍵字，讓工具幫你盯

小美的品牌最近在 Threads 上常被討論，有些是好評，但偶爾也有人抱怨出貨速度。她想自動追蹤這些討論，不用自己一則一則翻。

她打開 SocialBuddy 的設定頁面，在「關鍵字搜尋」區域新增幾組關鍵字：

- 品牌名稱（例如「美美小舖」）
- 「出貨慢」
- 「品質差」
- 競品品牌名稱

然後設定搜尋深度（往下滾幾次）和搜尋間隔（每 30 分鐘自動搜一次）。設好之後 SocialBuddy 就會定時到 Threads 上用這些關鍵字搜尋，搜到的貼文自動存起來。

**她做了什麼：** 在設定頁面輸入關鍵字，設定排程。
**SocialBuddy 做了什麼：** 定時自動搜尋，搜到的貼文自動備份，不用人盯著。

---

### 早上 10:00 — 客訴留言被刪了？沒關係

小美收到同事通知：「昨天那個客人在 FB 留了一則很長的負評，今天早上發現被刪掉了。」

以前碰到這種情況就沒轍了，留言刪了就是刪了。但現在小美打開 SocialBuddy 側邊面板，搜尋那則貼文，發現昨天瀏覽的時候已經自動備份了。不只文字，截圖也有。

她點開那則貼文的備份，完整的留言內容、留言時間、留言者帳號都在。就算對方已經把留言刪掉，她手上還是有完整的記錄。

**她做了什麼：** 在側邊面板搜尋該貼文。
**SocialBuddy 做了什麼：** 之前瀏覽時已自動備份，包含截圖，現在隨時可以調出來看。

---

### 下午 1:00 — 競品偷改價格，你有比對

小美注意到一家競品最近常常改文案和價格。上週她明明看到對方某產品寫 $899，今天再看變成 $699 了，但她也不確定是不是自己記錯。

她打開 SocialBuddy，找到那則貼文的備份。因為 SocialBuddy 會追蹤貼文的修改記錄，她可以看到這則貼文的「版本歷史」：

- **版本 1**（4/10 14:22）：文案寫「限時特惠 $899」
- **版本 2**（4/15 09:15）：文案改成「超值優惠 $699」

兩個版本的差異一目了然。文案哪裡改了、價格從多少變多少，SocialBuddy 會用顏色標出修改前後的不同。

**她做了什麼：** 點開貼文備份，切換不同版本查看。
**SocialBuddy 做了什麼：** 自動偵測到貼文內容改變，保留每次修改的版本，並標示差異。

---

### 下午 2:00 — 有人盜用你的商品圖

小美在 Threads 上發現有個帳號直接搬走了她們家的商品圖，連文案都抄了一半。她需要保留證據。

她瀏覽那則侵權貼文時，SocialBuddy 已經自動備份了。她到側邊面板找到那則貼文，點選「截圖」功能，SocialBuddy 自動把完整貼文畫面擷取下來，並且加上浮水印，標註截圖時間和網址。

這樣就算對方事後刪文，她手上有帶時間戳記的截圖，可以做為舉報的佐證。

**她做了什麼：** 瀏覽侵權貼文，在側邊面板點擊截圖。
**SocialBuddy 做了什麼：** 自動備份貼文、擷取截圖並加上時間浮水印。

---

### 下午 4:00 — 整理今天的巡邏報告

一天下來，小美備份了不少貼文。主管想看今天的巡邏整理，她需要快速產出一份報告。

她在側邊面板用日期和關鍵字篩選出今天的重點貼文，然後點「匯出」按鈕。SocialBuddy 提供多種匯出格式：

- **Excel（.xlsx）**：適合給主管看數據、做統計
- **HTML 報告**：有排版、有圖片，適合用瀏覽器打開閱覽
- **PNG 圖片包**：所有截圖打包下載
- **JSON / CSV**：需要進一步分析時用

小美選了 Excel 格式，匯出的檔案裡每則貼文一行，包含平台、作者、時間、內容摘要、有沒有被修改、截圖檔名等欄位。她把檔案丟到 Line 群組給主管，今天的巡邏報告就搞定了。

另外，她有在設定裡開「自動匯出」功能，每 60 分鐘自動存一份 JSON 備份到指定資料夾，就算 Chrome 突然當掉也不怕丟資料。

**她做了什麼：** 篩選貼文 → 點匯出 → 選格式 → 下載。
**SocialBuddy 做了什麼：** 把篩選後的貼文整理成報告檔案，格式齊全隨你挑。

---

### 下午 4:30 — 處理客訴，快速回覆

還有幾則客訴留言需要回覆。以前小美每次都要從頭想措辭、打字，碰到類似的問題還要翻之前的回覆來複製貼上。

現在她用 SocialBuddy 的「快速回覆」功能。她在側邊面板的貼文旁邊會看到回覆按鈕，點開後可以從 18 種預設範本中挑一個，範本有不同語氣分類：

- **溫暖親切**：「感謝您的回饋，我們非常重視每一位顧客的意見……」
- **專業穩重**：「您好，關於您提到的問題，我們已立即安排確認……」
- **澄清事實**：「感謝您的關注，關於這個部分，實際情況是……」

選好範本後可以再修改，確認沒問題就一鍵貼到留言框。如果預設範本不夠用，她也可以在設定頁面自己新增常用的回覆範本。

**她做了什麼：** 點回覆 → 選範本 → 微調 → 送出。
**SocialBuddy 做了什麼：** 提供分類好的回覆範本，一鍵帶入留言框，省去重複打字。

## 功能詳細說明

### 自動備份

瀏覽 Threads、Instagram、Facebook 時，SocialBuddy 會在背景自動偵測並儲存你看到的貼文。備份的內容包含：

- 貼文完整文字內容
- 作者帳號與顯示名稱
- 發布時間
- 留言數、按讚數等互動數據
- 貼文中的圖片（如有）
- 貼文的原始網址

備份是即時的，只要貼文出現在你的畫面上就會被記錄。所有資料儲存在你電腦的瀏覽器本機儲存空間中，不會上傳到任何伺服器。

### 關鍵字搜尋與排程

在設定頁面的「關鍵字」區域，你可以：

- 新增多組關鍵字
- 設定搜尋深度（往下滾動的次數，滾越多搜越深）
- 設定搜尋時間限制（超過設定時間自動停止）
- 開啟排程，讓 SocialBuddy 每隔一段時間自動執行搜尋

目前關鍵字搜尋功能支援 Threads 平台。搜尋到的貼文會自動備份到你的資料庫中。

### 版本追蹤

當你再次瀏覽到之前已備份的貼文時，SocialBuddy 會自動比對內容。如果貼文被修改過（文案改了、價格改了、圖片換了），系統會保留修改前後的所有版本。

在側邊面板中點開一則有修改記錄的貼文，你可以：

- 查看所有歷史版本的清單，每個版本都有時間戳記
- 切換不同版本查看完整內容
- 看到修改前後的差異標示（刪除的用紅色劃線，新增的用綠色標記）

### 自動截圖與浮水印

SocialBuddy 可以自動對貼文進行截圖，擷取的畫面包含完整的貼文內容、留言和圖片。截圖會自動加上浮水印，浮水印內容包含：

- 截圖時間（精確到秒）
- 貼文的原始網址

浮水印幫助你證明截圖的時間點，適合用來做為舉報、申訴的佐證資料。截圖會以 PNG 格式儲存，可以隨時匯出。

### 多格式匯出

SocialBuddy 支援多種匯出格式，滿足不同用途：

| 格式 | 副檔名 | 適合用途 |
|------|---------|----------|
| Excel | .xlsx | 給主管看報告、做數據統計 |
| HTML | .html | 有排版的視覺化報告，用瀏覽器打開 |
| JSON | .json | 程式開發者做進一步分析 |
| CSV | .csv | 匯入 Google Sheets 或其他試算表工具 |
| PNG | .png | 截圖圖片打包下載 |

你也可以設定自動匯出：在設定頁面指定匯出間隔（例如每 60 分鐘），系統會自動將資料備份到下載資料夾。這樣就算瀏覽器意外關閉，之前的資料也不會遺失。

匯出的檔案帶有數位簽章，你可以到 [官方驗證頁面](https://socialbuddy-verify.mitch.tw/verify) 上傳檔案驗證完整性，確認檔案沒有被事後竄改。

### 快速回覆範本

SocialBuddy 內建 18 種快速回覆範本，依照語氣和情境分類，方便你快速找到適合的回覆。

使用方式：在側邊面板的貼文旁邊點選回覆按鈕 → 選擇範本類別 → 選擇適合的範本 → 可以修改內容 → 一鍵貼到留言框。

如果預設範本不夠用，你可以在設定頁面自訂範本，也可以匯入其他語言的範本（到 [語言包](https://github.com/chenmitchell/socialbuddy-langpacks) 下載）。

### 資料安全與加密

SocialBuddy 所有資料都儲存在你自己的電腦上，具體來說是 Chrome 瀏覽器的本機儲存空間（IndexedDB）。沒有任何資料會上傳到雲端或經過任何伺服器。

安全功能包含：

- **本機加密**：可以在設定頁面開啟資料加密，用密碼保護你的備份資料
- **匯出檔驗證**：匯出的檔案會附帶數位簽章，可以到官網驗證檔案是否被竄改
- **不需註冊**：完全不需要建立帳號或登入，裝了就能用
- **不需付費**：完全免費，沒有任何隱藏收費

## 設定頁面說明

在 Chrome 工具列的 SocialBuddy 圖示上按右鍵 → 選項，可以打開設定頁面。設定頁面包含以下區域：

**一般設定**
調整基本的備份行為，例如是否自動截圖、截圖品質、浮水印樣式等。

**關鍵字管理**
新增、編輯、刪除搜尋關鍵字。每組關鍵字可以獨立設定搜尋深度和排程間隔。

**匯出設定**
選擇預設匯出格式、設定自動匯出的間隔時間和儲存位置。

**快速回覆**
管理回覆範本。可以新增自訂範本、編輯現有範本、匯入 / 匯出範本 JSON 檔案。

**安全性**
開啟或關閉本機加密、管理加密密碼。

## 常見問題

**Q：SocialBuddy 會不會蒐集我的個人資料？**
不會。所有資料都只存在你的電腦上，不會傳送到任何伺服器。SocialBuddy 不需要註冊帳號，也不會追蹤你的使用行為。

**Q：備份的資料存在哪裡？**
存在 Chrome 瀏覽器的本機儲存空間（IndexedDB）。如果你解除安裝擴充功能，這些資料會一起被清除。建議定期使用匯出功能備份到電腦的資料夾中。

**Q：可以備份私人帳號或私人社團的貼文嗎？**
SocialBuddy 只會備份你在瀏覽器中實際看到的內容。如果你有權限瀏覽某則貼文，它就能被備份。SocialBuddy 不會繞過任何平台的權限限制。

**Q：為什麼只有 Threads 支援關鍵字搜尋？**
因為 Threads 的搜尋功能是公開的網頁介面，SocialBuddy 可以透過自動化的方式在上面搜尋。IG 和 FB 的搜尋機制不同，目前暫不支援。

**Q：貼文被刪掉了，SocialBuddy 還能備份到嗎？**
如果貼文在被刪除之前你已經瀏覽過，或者透過關鍵字搜尋已經抓到，那備份就在你手上了。但如果貼文在你看到之前就被刪除，SocialBuddy 是無法備份的。

**Q：匯出的 Excel 檔案可以驗證嗎？**
可以。匯出的檔案帶有數位簽章，到 [官方驗證頁面](https://socialbuddy-verify.mitch.tw/verify) 上傳檔案，就能驗證內容是否被竄改過。

**Q：我可以在多台電腦上使用嗎？**
可以，每台電腦各自安裝就好。但因為資料是存在本機的，各台電腦的資料是獨立的，不會自動同步。你可以用匯出功能把資料從 A 電腦搬到 B 電腦。

**Q：支援哪些語言？**
擴充功能內建繁體中文與英文。如果需要其他語言，可以到 [語言包](https://github.com/chenmitchell/socialbuddy-langpacks) 下載，目前有簡體中文、日本語、한국어、Español、Português。
