## 這是什麼？

你是社群小編，每天要盯好幾個平台的留言和貼文？

SocialBuddy 幫你把看到的貼文自動存起來，不管對方改了還是刪了，你手上都有備份。關鍵字搜尋、自動截圖、一鍵匯出報告 — 該有的都有，而且**完全免費、資料不上雲**。

## 能幫你做什麼？

**📋 自動備份**
瀏覽 Threads、IG、FB 的時候自動幫你存，不用手動截圖了。

**🔍 關鍵字搜尋**
設好關鍵字，讓它自己去 Threads 上搜，搜到的自動存下來。可以設搜多深、搜多久。

**📸 自動截圖**
每則貼文自動截圖，含留言、圖片，還會加上浮水印，存證用剛剛好。

**📤 多格式匯出**
JSON、CSV、HTML、Excel、PNG，想要什麼格式自己勾。還可以設定每幾分鐘自動匯出一次，不怕當掉丟資料。

**💬 快速回覆**
18 種預設回覆範本，切個語氣選一則，一鍵貼到留言框，省下打字時間。

**🔒 你的資料你作主**
所有東西都存在你自己的電腦裡，不經過任何伺服器。要加密？有。要驗證檔案沒被改過？也有。

## 怎麼用？

> 客人留負評結果改掉刪掉、有人冒用你的圖你要截圖舉報他已經下架了、競品偷改價改文案你想比對什麼都沒留下。裝了 SocialBuddy 之後，這些通通有備份。

## 安裝方式

### Chrome Web Store（推薦）

*即將上架，敬請期待。*

### 手動安裝

1. [下載最新版 ZIP](https://github.com/chenmitchell/social-buddy-release/raw/main/socialbuddy-v1.0.5.zip)
2. 解壓縮到一個資料夾
3. Chrome 打開 `chrome://extensions/`
4. 右上角開「開發人員模式」
5. 點「載入未封裝項目」，選解壓縮出來的那個資料夾
